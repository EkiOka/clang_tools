# C言語開発ツール群

個人で利用するC言語の開発ツール群です。
主な目的はvscodeでの開発支援となります。

## 対応機能

対応する機能は以下の通りとなります。

* Doxygenのドキュメント生成
* Doxygen、および静的解析ツール結果ををYAMLに変換
* YAMLのデータを検証してのコーディング規約チェック
    * チェック結果の逸脱箇所の許容理由の保存
    * 前回逸脱箇所との比較、許容理由は前回値適応する形にしてチェック数を減らす。
    * 構成管理にソースコードと共に登録して品質管理データとして使えるようにしたい。
